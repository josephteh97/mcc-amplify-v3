  ● Three ways, depending on what you're doing:                                                                                                                                                                    
                                                                                                                                                                                                                 
  1. Fix a wrong bbox (edit mode)                                                                                                                                                                                
  Select the detection row → click P1 on image → click P2 → auto-saved immediately as confirm
                                                                                                                                                                                                                 
  2. Mark a false positive                                                                                                                                                                                       
  Select the row → set Judgement = "wrong" → 💾 Save Correction → saved as reject                                                                                                                                
                                                                                                                                                                                                                 
  3. Add a missed column                                                                                                                                                                                         
  Make sure NO row is selected → click P1 → click P2 → set Shape → ➕ Save New Detection → saved as add                                                                                                          
                                                                                                                                                                                                                 
  All three call agent.add_correction(tile_hash, action, shape, bbox_norm) where bbox_norm is the bbox as 0–1 floats relative to tile size. These get injected into the moondream prompt on the next detection   
  run as:                                                                                                                                                                                                        
                                                                                                                                                                                                                 
  Prior human corrections — REAL columns: square@[0.12,0.34,0.18,0.41] | FALSE positives to skip: round@[0.5,0.5,0.55,0.6]
                                                                                                                                                                                                                 
  The key rule: always detect the tile first before saving corrections, so the tile_hash is set. The hash ties your corrections to the exact tile content — if you save without a hash, nothing gets stored.
