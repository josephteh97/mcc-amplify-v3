1. Load — enter the PDF path, click 📂 Load → tiles are prepared                                                                                                                                                                                                                                                                                                                                                                  
  2. Detect — click 🔍 Detect tile → moondream runs on the current tile, boxes appear on the image                                                                                                                                                                                                                                                                                                                                  
  3. Review — for each detection in the table:                                                                                                                                                                                                                                                                                                                                                                                      
    - Pick the # (detection number)                                                                                                                                                                                                                                                                                                                                                                                                 
    - Click ✅ Correct → saved as confirm to memory.json + SQLite immediately                                                                                                                                                                                                                                                                                                                                                       
    - Click ✗ Wrong → saved as reject, cross drawn through the box                                                                                                                                                                                                                                                                                                                                                                  
  4. Draw correction — hover over the tile image to read pixel coordinates (shown bottom-left on hover), then:                                                                                                                                                                                                                                                                                                                      
    - Enter x1, y1, x2, y2                                                                                                                                                                                                                                                                                                                                                                                                          
    - Click 👁 Preview → see the blue box before saving                                                                                                                                                                                                                                                                                                                                                                              
    - Click 💾 Save Correction → saved as add to memory                                                                                                                                                                                                                                                                                                                                                                             
  5. Navigate — ◀ Prev / Next ▶ to move between tiles                                                                                                                                                                                                                                                                                                                                                                               
  6. Next run — re-detect the same tile and the corrections appear in the prompt: the LLM sees "Human confirmed X at [bbox], rejected Y as false positive, added Z at [bbox]"
